# Pomodoro Timer — 极简桌面番茄钟

## 概述

一个基于 Electron 的极简桌面番茄钟，运行于系统托盘，通过小浮窗交互。无持久化存储，用完即走。

## 技术栈

- **框架**: Electron
- **语言**: HTML + CSS + JavaScript (Vanilla)
- **无外部依赖** (无框架、无数据库、无构建工具)

## 核心功能

1. **专注计时** — 25 分钟倒计时
2. **休息计时** — 5 分钟倒计时（专注结束后自动进入）
3. **开始/暂停/重置/停止** 控制
4. **计时完成通知** — 系统通知 + 提示音
5. **托盘图标状态指示** — 颜色随状态变化

## 状态机

┌─────────┐  ▶ 开始专注  ┌─────────┐  倒计时归零  ┌─────────┐
│  Idle   │ ──────────► │  Focus  │ ──────────► │  Break  │
│  (空闲)  │             │  (专注)  │             │  (休息)  │
│         │ ◄────────── │  25min  │             │  5min   │
└─────────┘  ⏹ 停止    │  红色   │  倒计时归零  │  绿色   │
                          └─────────┘             └─────────┘
                              ↑    ↺ 重置           │
                              └──────────────────────┘
                                     跳过休息

### 状态说明

- **Idle**: 初始状态，显示 25:00，按钮显示"▶ 开始"
- **Focus**: 25min 倒计时，红色主题，环形进度递减，托盘图标红色
- **Break**: 5min 倒计时，绿色主题，环形进度递减，托盘图标绿色
- 任何状态下点击 ⏹ 回到 Idle，点击 ↺ 重置当前计时

## 架构

```
Electron App
│
├── Main Process
│   ├── main.js          — 入口，创建托盘和窗口
│   ├── timer.js         — 计时器逻辑（setInterval，状态机）
│   └── notification.js  — 系统通知 + 提示音播放
│
└── Renderer (浮窗)
    ├── index.html       — 浮窗 DOM
    ├── style.css        — 样式（暗色主题，环形进度动画）
    └── renderer.js      — UI 更新、按钮事件 → IPC 发送到 Main
```

### 进程通信

```
Main → Renderer: timer:tick (剩余秒数, 状态)
Main → Renderer: timer:complete (状态切换)
Renderer → Main: timer:start, timer:pause, timer:reset, timer:stop
```

## UI 规格

- **窗口尺寸**: 320 × 220px
- **窗口属性**: 无边框 (frameless)，可拖拽，置顶
- **打开方式**: 点击托盘图标显示/隐藏
- **关闭行为**: 关闭窗口 = 隐藏到托盘（不退出）

- ─────────

## 边界情况

- **计时中关闭浮窗**: 计时不中断，托盘图标持续指示状态
- **系统通知被拒绝**: 静默降级，不影响计时
- **提示音播放失败**: 静默降级，不影响计时
- **专注→休息过渡**: 自动切换，无需用户操作
- **休息结束**: 回到 Idle，等待用户手动开始下一轮
